# Pipeline Map

This is an honest map of the project pipeline as represented by this
GitHub code repository. The actual work was iterative: prompts, labeling functions, sample splits,
QA reports, and extraction runs were repeatedly checked and revised. This repository
does not pretend the project was a single clean command. It documents the real
stages and the restricted artifacts that connect them.

All `data/*`, `codex_outputs/*`, `docs/figures/*`, and `paper/build/*` artifacts
mentioned below are local outputs generated by credentialed users from their own
MIMIC access. They are not shipped.

## 1. Database Exploration and Schema Sanity Checks

Scripts:

- `scripts/01_explore_mimic_notes.py`

Restricted input:

- `<MIMIC_ROOT>` loaded into PostgreSQL and configured by `MIMIC_PG_URI`.
- MIMIC-IV-Note discharge notes and MIMIC-IV hosp admissions/diagnosis tables.

Produces:

- `codex_outputs/01_mimic_exploration.md`

Feeds:

- Human confirmation that `hadm_id` joins are usable, note lengths are plausible,
  and ICD distributions look reasonable.

Messy/iterative note:

- This exploratory report can include short note excerpts if run as written. It is
  a local DUA-restricted QA artifact and must not be published.

## 4. Extraction Runs

Scripts:

- `scripts/05_run_smoke_coverage.py`
- `scripts/05_validate_on_holdout.py`
- `scripts/06_run_1k_smoke.py`
- `scripts/07_run_5k_methodology.py`
- `scripts/08_run_full_production.py`
- `scripts/31_launch_production.sh`
- `scripts/32_monitor_production.py`
- `scripts/40b_retry_failed.py`
- `scripts/45_run_gold_standard.py`
- `scripts/55_run_paired_gold.py`
- `scripts/56_run_reasoning_comparison.py`
- `scripts/46_run_extended_5k.py`

Restricted input:

- `data/splits/*.csv`
- MIMIC discharge-note text fetched by `hadm_id`
- OpenAI API access configured by `OPENAI_API_KEY`

Produces:

- `data/raw_responses/<run_id>/*.json`
- `data/raw_responses/<run_id>/results.jsonl`
- run-cost logs under local `logs/`
- extraction summaries under `codex_outputs/`

Variant/model-size structure:

- Prompt variants A/B/C were run on refinement, holdout, methodology, and audit
  samples.
- Paired gold/model-size analyses compare smaller and larger model outputs over
  matched samples.
- Reasoning-instruction analyses compare outputs with and without explicit
  reasoning instructions.

Messy/iterative note:

- Runs were not one pass. There were prompt dry runs, smoke coverage checks,
  holdout checks, failed-call retry lists, production monitoring, post-run QA,
  and targeted reruns before final aggregate analyses were produced.
